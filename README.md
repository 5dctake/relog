# ReLog

Review × Log｜人生をレビューする日記

「とりあえず星つけとく」くらいの気軽さで、今日あったことに星をつけて保存するだけ。
人生の些細な瞬間をレビューし、あとから見返すためのシンプルな日記アプリです。

## 主な機能

- **クイック入力**: タイトルと星だけで30秒以内に保存可能。
- **カレンダー連携**: 過去の記録の日付も自由に変更可能。
- **気分スタンプ**: 5種類のスタンプでその時の気分を記録。
- **月末レポート**: 1ヶ月の振り返りを自動集計。平均評価やカテゴリ別分析。
- **バックアップ**: JSON形式でのデータエクスポート・インポート。
- **フィルタ・検索**: 評価やカテゴリ、気分での詳細な絞り込み。

## 技術スタック

- **Framework**: Flutter
- **State Management**: Riverpod
- **Local Database**: Isar (Stable configuration)
- **Ads**: Google Mobile Ads (Interstitial)
